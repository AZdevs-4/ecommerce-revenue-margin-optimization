# Executive Power BI Sales & Profit Performance Analytics Dashboard

## 📝 Project Description
Many businesses with high sales volumes operate in a blind spot because their actual profits are eaten away by unmonitored discounting habits and logistics costs. This Power BI project tracks $79.79M in Net Revenue and transforms messy transactional data into an interactive executive canvas to protect final net profits.

## ❓ The Core Business Questions Answered
* **"Where exactly are our value leaks, and how are costs distributed?"** (Answered via the Total Investment Distribution tracker, which maps operational expenses like $4.13M in discounts and $1.25M in shipping directly against margins).
* **"Which cities and product categories are our true profit champions?"** (Answered via dynamic charts showing that Islamabad is the top-performing market bringing in 29.32% of all net profit, while Electronics is the highest-earning category at $7.19M).
* **"Are we hitting our target profit margin right now?"** (Answered instantly via the central target gauge chart showing an exact corporate baseline margin of 21.21%).

## ⚡ Technical Features (Built in Power BI)
* **Unified Relational Data Modeling:** Designed an optimized data model connecting localized dimension tables (`Dim_City`, `Dim_Product`) to a core transaction table.
* **AI-Powered Smart Narrative Engine:** Programmed a dynamic text-summary block that auto-generates written performance briefs in plain English.
* **Multi-Layered Interactive Slicers:** Created responsive filters for Months, Product Categories, and key regions.
* **Product Volume Funnel Visuals:** Built a structured product volume funnel that automatically ranks inventory sales from top-tier revenue drivers.
